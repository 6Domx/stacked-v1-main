# Stacked Portfolio

A simple web app using Flask (backend) and Tailwind CSS (frontend).

## Prerequisites

- **Python** (>= 3.8)

## Installation

1. Clone the repository

2. Install Flask:
   ```bash
   pip install flask

3. Run the application:
   ```bash
   python app.py

4. Open http://127.0.0.1:5000 in your browser.
